# TrustPoint CPA Template

A professional, trust-focused single-page template ideal for solo practitioners.

**Tier:** Budget ($500-$750)  
**Type:** Single-page with personal branding focus

## Features

- ✅ Trust-building hero with stats
- ✅ Trust badges section
- ✅ 6 service cards with icons
- ✅ Personal "Meet Your CPA" section
- ✅ Client testimonials
- ✅ Multiple contact methods
- ✅ Contact form
- ✅ Responsive design
- ✅ CSS variables for branding

## Quick Start

1. Edit colors in `css/styles.css`
2. Replace content in `index.html`
3. Add professional headshot to `assets`
4. Update contact information

## Customization Guide

### Colors
```css
:root {
    --color-primary: #0f4c81;   /* Navy blue */
    --color-accent: #f5a623;    /* Gold */
}
```

### Content to Replace
- [ ] Company name and logo icon
- [ ] Hero headline and stats
- [ ] Trust badges (certifications)
- [ ] Service descriptions
- [ ] CPA bio and photo
- [ ] Education and certifications
- [ ] Testimonials
- [ ] Contact information

### Personal Touch
This template is designed for solo practitioners. Customize the "Meet Your CPA" section with:
- Professional headshot
- Personal bio
- Credentials and education
- Years of experience
