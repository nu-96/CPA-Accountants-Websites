# Nexus Partners - Premium Template

A modern, tech-forward premium template for startup-focused CPA and advisory firms.

**Tier:** Premium ($2,000-$2,500+)  
**Type:** Full-featured single-page with dark sections

## Design Features

- ✅ Startup/VC-friendly aesthetic
- ✅ Dark mode hero section
- ✅ Transparent navbar that fills on scroll
- ✅ Industry expertise cards
- ✅ Team section with LinkedIn integration
- ✅ Dark contact section with glassmorphism form
- ✅ Smooth scroll navigation
- ✅ AOS animations throughout

## Unique Elements

- Indigo/cyan color palette (tech-forward)
- Animated background blobs
- Scroll indicator animation
- Team card hover effects with social links
- Industry emoji icons for approachability

## Technologies

- Tailwind CSS
- AOS library
- Custom CSS animations
- CSS backdrop-filter

## Customization

### Colors
```css
:root {
    --color-primary: #6366f1;   /* Indigo */
    --color-accent: #22d3ee;    /* Cyan */
    --color-dark: #0f172a;      /* Slate */
}
```

### Single Page to Multi-Page

This template is a single-page design. To expand:
1. Create separate HTML files for each section
2. Update navigation links
3. Add consistent headers/footers
