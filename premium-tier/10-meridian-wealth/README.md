# Meridian Wealth - Premium Template

An ultra-elegant, luxury template for high-net-worth private client and family office services.

**Tier:** Premium ($2,000-$2,500+)  
**Type:** Single-page luxury design

## Design Philosophy

This template embodies understated elegance with:
- Serif headings (Cormorant Garamond)
- Warm cream background
- Gold/bronze accents
- Generous whitespace
- Refined typography

## Premium Features

- ✅ Luxury typography (Google Fonts)
- ✅ Asymmetric hero layout
- ✅ Floating stat card animation
- ✅ Split-background design
- ✅ Numbered service list
- ✅ Large statistics display
- ✅ Editorial-style insights section
- ✅ AOS animations throughout

## Customization

### Colors
```css
:root {
    --color-primary: #1a2e3b;   /* Deep blue-gray */
    --color-accent: #b8926a;    /* Gold/bronze */
    --color-cream: #f8f7f4;     /* Warm white */
}
```

### Typography
Fonts are loaded from Google Fonts:
- Cormorant Garamond (headings)
- Montserrat (body)

For production, consider self-hosting fonts.

### Content Tone
This template uses formal, refined language appropriate for:
- "Distinguished families"
- "Discerning clients"
- "Private consultation"

Adjust terminology for your market.
