# Heritage Advisors Template

An elegant, family-focused template for wealth management and estate planning firms.

**Tier:** Mid ($1,000-$1,500)  
**Type:** Multi-page with elegant design

## Pages Included

1. **index.html** - Elegant homepage with family focus
2. **services.html** - Estate, tax, trust, and advisory services
3. **approach.html** - Philosophy and process
4. **team.html** - Leadership and advisor profiles
5. **contact.html** - Private consultation request

## Features

- ✅ Elegant serif typography (Playfair Display)
- ✅ Warm, trustworthy color palette
- ✅ Family-focused messaging
- ✅ Generational stats and trust indicators
- ✅ Featured testimonial
- ✅ Team hierarchy display
- ✅ Private consultation request form
- ✅ Mobile responsive

## Customization

### Colors
```css
:root {
    --color-primary: #2c4a3e;   /* Forest green */
    --color-accent: #c9a962;    /* Gold */
}
```

### Fonts
Uses Google Fonts:
- Playfair Display (headings)
- Source Sans Pro (body)

Replace by updating the Google Fonts link and CSS variables.
